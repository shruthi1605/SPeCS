SPeCS - Student Project eCatalogue System - is a portal that enables the student community to showcase their academic IT project experience to the hiring managers.

This portal stores all projects details of a student that he/she has undertaken either part of academic curriculum or driven by passion to develop IT applications.

If you are a first time student user to this portal, kindly signup to create your profile and add your project details.

If you a hiring manager, you can search on a term - student name, project title, domain, technology, programming language etc - to view the project details of the application details developed by the student. While viewing the project details, kindly remember to RATE the project as well.
